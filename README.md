# Automated Content Generation

This project use OpenAI, Stable Diffusion (Replicate.com), GridDB, and Node.js

## Run the project

Clone this repository

```
git clone https://github.com/junwatu/automated-content-generation.git

cd automated-content-generation

npm install
```

Rename `.env.example` to `.env` and then set the API keys:

```
OPENAI_API_KEY=
OPENAI_API_ORG=
REPLICATE_API_KEY=
```

Run the project

```
npm run dev
```
